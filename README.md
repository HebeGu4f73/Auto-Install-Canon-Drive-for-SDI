# Auto-Install-Canon-Drive-for-SDI
Automatically install the driver for the Shude International Campus Printer for Windows

## 最新版本介绍 - v1.0.0
【by School Printer Technical Improvement Group】

2026/03/23 Update

### 功能说明 | Function Declaration
本工具实现了 Microsoft Windows (10/11) 版的 Canon Generic Plus UFR II/LIPSLX/PCL6/PS3 打印机驱动程序 3.31 版本自动化安装；

自动化配置成都树德中学学生打印机；

【注意：在成都树德中学局域网内才能够正常配置并使用】

### 使用指南 | User Guide
请下载release中的压缩包文件

解压并直接运行文件夹中 autodrive.exe 文件，界面友好开袋即食~
【注意：删改其它文件将导致软件功能失效】
